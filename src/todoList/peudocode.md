# TO-DO List App

## Requirements
- An input/form that accepts to-do task from the user.
- A button that adds the tasks to the existing to-do list
- A section that displays the whole task list.
- An icon on each task that, when clicked, removes the task from the list.
- A react state for persisting the tasks in memory.
- A local storage for persisting the tasks across sessions.
- An icon library.

#### Steps

1. Create the skeletal structure of the app
2. Build modules/reusable components
3. Create React states
